## Overview

This page contains a curated collection of tools, resources, and references I use in my cybersecurity work. These are organized by category to help you find what you need quickly.

---

## Digital Forensics & Incident Response

### Analysis Tools

**[Autopsy](https://www.autopsy.com/)**  
Open-source digital forensics platform for disk and memory analysis.  
*Use case: File system analysis, timeline creation, keyword searches*

**[FTK Imager](https://www.exterro.com/ftk-imager)**  
Forensic imaging tool for creating disk images and previewing evidence.  
*Use case: Evidence acquisition, image verification, file extraction*

**[Volatility](https://www.volatilityfoundation.org/)**  
Advanced memory forensics framework for analyzing RAM dumps.  
*Use case: Memory analysis, malware detection, process investigation*

**[Sleuth Kit](https://www.sleuthkit.org/)**  
Collection of command-line tools for analyzing disk images.  
*Use case: File system analysis, deleted file recovery, timeline analysis*

### Evidence Collection

**[KAPE](https://www.kroll.com/en/services/cyber-risk/incident-response-litigation-support/kroll-artifact-parser-extractor-kape)**  
Kroll Artifact Parser and Extractor for rapid evidence collection.  
*Use case: Triage collection, artifact parsing, automated analysis*

**[Velociraptor](https://www.rapid7.com/products/velociraptor/)**  
Advanced endpoint visibility and collection tool.  
*Use case: Live response, hunting, artifact collection*

## Network Security

### Traffic Analysis

**[Wireshark](https://www.wireshark.org/)**  
Industry-standard network protocol analyzer.  
*Use case: Packet capture, protocol analysis, network troubleshooting*

**[tcpdump](https://www.tcpdump.org/)**  
Command-line packet analyzer for Unix-like systems.  
*Use case: Network traffic capture, filtering, analysis*

**[NetworkMiner](https://www.netresec.com/?page=NetworkMiner)**  
Network forensic analysis tool for passive network sniffing.  
*Use case: File extraction from PCAP, OS fingerprinting, credential detection*

### Scanning & Reconnaissance

**[Nmap](https://nmap.org/)**  
Network discovery and security auditing tool.  
*Use case: Port scanning, service detection, network mapping*

**[Masscan](https://github.com/robertdavidgraham/masscan)**  
Fast TCP port scanner for large-scale networks.  
*Use case: Large network scans, internet-wide surveys*

## SIEM & Log Analysis

### SIEM Platforms

**[ELK Stack](https://www.elastic.co/elastic-stack)**  
Elasticsearch, Logstash, and Kibana for log management and analysis.  
*Use case: Log aggregation, searching, visualization, alerting*

**[Wazuh](https://wazuh.com/)**  
Open-source security platform for threat detection and response.  
*Use case: Security monitoring, compliance, file integrity monitoring*

**[Splunk](https://www.splunk.com/)**  
Enterprise platform for searching, monitoring, and analyzing data.  
*Use case: Log analysis, security monitoring, incident investigation*

## Web Application Security

### Proxy & Interception

**[Burp Suite](https://portswigger.net/burp)**  
Web application security testing platform.  
*Use case: Intercepting proxy, vulnerability scanning, manual testing*

**[OWASP ZAP](https://www.zaproxy.org/)**  
Open-source web application security scanner.  
*Use case: Automated scanning, manual testing, API testing*

### Directory & File Discovery

**[Gobuster](https://github.com/OJ/gobuster)**  
Fast directory and DNS busting tool.  
*Use case: Directory enumeration, DNS subdomain discovery*

**[Dirb](https://github.com/v0re/dirb)**  
Web content scanner for finding hidden files and directories.  
*Use case: Web server enumeration, content discovery*
