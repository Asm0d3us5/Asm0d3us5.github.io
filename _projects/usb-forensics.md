## Overview
This project investigates a USB drive related to a computer fraud case, 
focusing on identifying relevant artifacts, 
recovering deleted files, 
and validating digital evidence using forensic techniques.

## Tools Used
- Autopsy  
- Foremost  
- FTK Imager  

## What I Learned
- File recovery techniques and carving deleted data from storage media  
- Metadata analysis to establish timelines and understand file activity  
- Evidence validation using cryptographic hashes (MD5/SHA-256) to ensure data integrity and maintain forensic soundness
